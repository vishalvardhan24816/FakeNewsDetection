{{ brand }} runs five independent gates on a news headline and only marks it as real if every single one passes.

Check a headline
1. Spelling

NER-aware spell check. Skips named entities, fails the headline if the typo ratio is too high.

2. Clickbait

A custom-trained classifier flags sensationalised, vague or attention-bait headlines.

3. Subjectivity

A transformer model rejects opinion-style headlines that don't read like real reporting.

4. News-title shape

A second classifier rejects normal sentences that don't match the structure of a real news title.

5. Web presence

Searches Google, computes sentence similarity against real news results, and asks an LLM to confirm the match.
