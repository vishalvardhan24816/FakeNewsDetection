About {{ brand }}

A 5-step rule-based fake news detector. Originally a 2023 college project; rebuilt from scratch as a small modular Python package.

How it works

Most fake-news projects throw a single ML model at the headline and call it a day. {{ brand }} takes the opposite approach: a real news headline should pass five independent sanity checks, each of which catches a different failure mode.

  1. Spelling. Real journalism is edited; fake stories often contain obvious typos.
  2. Clickbait. A trained classifier rejects attention-bait phrasing.
  3. Subjectivity. A transformer rejects opinion-style writing that's not actually reporting.
  4. News-title shape. Another classifier rejects full sentences that don't look like a headline.
  5. Web presence. Searches Google, finds semantically similar headlines from real news domains, and asks an LLM whether they're saying the same thing.

If any check fails, the headline is flagged. The result page shows you exactly which gate it tripped on.

Limitations

  • The web-presence check scrapes Google with selectors that are fragile and may break when Google updates its HTML.
  • The Anthropic API is required for the final gate.
  • Don't act on the verdict alone — always cross-check with reputable sources before sharing or publishing.
