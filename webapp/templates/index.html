5-step validation pipeline

Don't share it
before you verify it.

{{ brand }} runs five independent gates — spelling, clickbait, subjectivity, news-title shape, and live web verification — and only marks a headline real if every single one passes.

Verify a headline How it works
01
Spelling

NER-aware spell check. Skips named entities, fails the headline if the typo ratio is too high.

02
Clickbait

A custom-trained classifier flags sensationalised, vague or attention-bait headlines.

03
Subjectivity

A transformer model rejects opinion-style headlines that don't read like real reporting.

04
News-title shape

A second classifier rejects normal sentences that don't match the structural shape of a real news title.

05
Web presence

Searches the web for matching coverage from trusted news domains and asks an LLM to confirm the match.
